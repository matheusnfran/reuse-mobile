# ReUse! ♻️

Aplicativo mobile de marketplace para reutilização e troca de itens, desenvolvido em React Native com Expo.

## Sobre o Projeto

O ReUse! é uma plataforma mobile que conecta pessoas interessadas em dar uma nova vida a objetos usados, promovendo o consumo consciente e a sustentabilidade.

## Telas

- **Login** — autenticação do usuário
- **Home** — listagem de itens disponíveis com filtros por categoria e favoritos
- **Produto** — detalhes do item selecionado
- **Perfil** — informações do usuário e seus itens cadastrados

## Funcionalidades

- Navegação entre telas com Expo Router
- Favoritar itens com persistência via **Async Storage**
- Foto de perfil tirada diretamente pela **câmera** do dispositivo
- Filtros de categoria na tela inicial
- Layout responsivo com identidade visual sustentável

## Tecnologias Utilizadas

- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
- [Expo Router](https://expo.github.io/router/)
- [Async Storage](https://react-native-async-storage.github.io/async-storage/)
- [Expo Image Picker](https://docs.expo.dev/versions/latest/sdk/imagepicker/)

## Como Rodar o Projeto

```bash
npm install
npx expo start
```

Escaneie o QR code com o app **Expo Go** no celular para visualizar.
